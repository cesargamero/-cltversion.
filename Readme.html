Gerenciamento de Configuração de Software

O gerenciamento de configuração de software é uma prática de engenharia de sistemas que visa acompanhar e controlar as mudanças nos componentes e nas configurações de sistemas de software. No desenvolvimento de software, essa prática é frequentemente combinada com o controle de versões e a infraestrutura de integração contínua e entrega contínua (CI/CD).

Controle de mudanças

Controle de mudanças: Envolve testar as modificações assim que são feitas para garantir que tudo funcione corretamente.

Integraçao continua

Integração contínua: Monitora e registra a versão atual do software e as particularidades de cada versão.

Controle de versão

Controle de versão: Registra a versão atual do software e as especificações associadas a cada versão.
